<h1>Задание:</h1>
Добавить в Solution с проектами для работы с банковским счетом (Homework Day 15) проект ASP.NET MVC.<br>

Для web-приложения реализовать следующие возможности:<br>

* выполнить пополнение на счет;<br>

* выполнить списание со счета;<br>

* выполнить перевод с одного счета на другой, в случае принадлежности счетов как одному владельцу, так и различным;<br>

* создать новый счет;<br

* закрыть счет.<br>

В логику работы приложения добавить функциональную возможность рассылки владельцу счета сообщений с информацией о списанной или пополненной сумме на его счету, а также о начисленных бонусных баллах.
<br>
Для стилизации web-приложения использовать Bootstrap css-фреймворк.<br>
